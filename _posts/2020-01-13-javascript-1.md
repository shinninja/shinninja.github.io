## JavaScript란?

* 웹 브라우저를 제어하기 위한 언어

## 탈웹

* Google Apps Script (구글 스프레드시트)

## 언어란?
* 의사소통을 위한 <strong>약속</strong>
* 언어를 사용하는 대상 : 환경
* JavaScript  ~('Hello world') 이러한 언어를 호출 했을때,<br>
  각각 다른 환경에 맞는 명령어를 사용해서  'Hello world' 라는 메시지를 띄운다.


  * 웹브라우저 = alert 
  * Node.js = write
  * SpreaSheet = msgBox
